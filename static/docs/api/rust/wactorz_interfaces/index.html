§wactorz-interfaces

Human and machine interfaces for AgentFlow.

  • cli — interactive REPL / command-line interface
  • rest — axum HTTP REST API
  • ws — WebSocket bridge: MQTT ↔ browser clients (for the Babylon.js dashboard)
